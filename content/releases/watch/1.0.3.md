# Release 1.0.3 — App del reloj

Primera actualización desde la **1.0.2** (6 de marzo de 2026). Mejoras de protocolo, comunicación con el teléfono y serialización.

## ✨ Cambios

- **Migración de Gson a Kotlinx Serialization** para todo el manejo JSON: arranque más rápido, menos overhead en runtime y mayor compatibilidad con el backend.
- **Protocolo PING/PONG** entre reloj y teléfono con utilidades de testing dedicadas.
- **Mediciones bajo demanda**: el reloj ahora atiende solicitudes puntuales de medición de salud desde el teléfono (`WatchReceiverService`).
- **HealthSyncService**: logging de diagnóstico detallado en el sincronizador periódico y el resumen diario para depurar problemas de sincronía.
- **Notificaciones simplificadas**: se removió el manejo de sonido de alarma del `HabitService`.

## ✅ Compatibilidad

- Compatible con la app móvil **1.0.5** y con el backend actualizado (`/health/patient/{id}/summary` corregido).
- Sin cambios destructivos: instalable sobre la 1.0.2.
